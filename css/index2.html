HTML é a lingugagem padrao para construcao de páginas web.

Com HTML, é possivel definir o conteudo e a estruturação
do documento

Tags HTML permitem a criação
de links, imagem e tabelas

Java Script é uma linguagem de programao usada para criar interatividade na web

CSS é uma linguagem usada para estilizar documentos HTML.

Com CSS, é possivel controlar aparência e o layout da página.
